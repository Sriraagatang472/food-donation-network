A web-based system that connects food providers (restaurants, homes, hostels) with nearby NGOs for efficient food donation.

## Features
- Donation form with input validation
- City-based NGO assignment
- Quantity-based NGO allocation
- Backend data storage using PHP
- NGO dashboard to view requests
- Organized display of donations

## Technologies Used
- HTML, CSS, JavaScript
- PHP
- Apache Server
- Kali Linux

## How It Works
1. Food provider fills the donation form
2. System assigns NGO based on location and quantity
3. Data is stored in the backend
4. NGO can view requests in dashboard
